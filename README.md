# 🧠 Brain Tumor Detection using CNN

This project is a deep learning-based system for detecting brain tumors from MRI images using Convolutional Neural Networks (CNN).

---

## 🚀 Features
- Image preprocessing using OpenCV
- CNN model for classification
- Accuracy and loss visualization
- Evaluation using classification report & confusion matrix
- Predict tumor presence from new MRI images

---

## 📂 Dataset
- Brain MRI Images (Tumor vs No Tumor)
- Binary classification problem

---

## 🧠 Model Architecture
- Conv2D → MaxPooling
- Conv2D → MaxPooling
- Flatten → Dense → Dropout → Output

---

## 📊 Results
- Achieved good accuracy on test dataset
- Evaluated using:
  - Precision
  - Recall
  - F1-score
  - Confusion Matrix

---

## 🛠️ Tech Stack
- Python
- TensorFlow / Keras
- OpenCV
- NumPy
- Matplotlib
